Git

  • git init - Создает новый локальный репозиторий Git в текущей директории
  • git clone <..URL..> - Клонирует удаленный репозиторий (например, с GitHub) на ваш компьютер
  • git add <файлы> - Добавляет изменения в рабочей директории в «индекс» (staging area) для последующего коммита
  • git commit -m "сообщение" - Фиксирует изменения из индекса, создавая новый коммит с описанием изменений
  • git push - Отправляет локальные коммиты в удаленный репозиторий (например, на GitHub)
